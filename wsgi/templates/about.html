This is a free web app developed & maintained by Ashish Nitin Patil.
It is exclusive to Mu Sigmans (official email id needed to register).

Disclaimer: Mu Katta is NOT an official website of Mu Sigma!

Mu Katta is a simple attempt at easing the following problems...

  • Want to share a cab to airport?
  • Hunting for a room-mate or a new place to move in?
  • Want to carpool to work? (conceptual stage... hop in!)

Technical details...

The web app is powered by the Open Shift cloud platform by Red Hat.
It runs on Python & uses the Django web framework.
The Bootstrap 3 & django-bootstrap3 utilities make the site "pretty".
Contact me if you need any further info!

Privacy & TnC...

Even though the website code is Open Sourced, it is very secure.
No third party (non Mu Sigma email ids) can access website data.
No, I cannot access your plaintext passwords.
For legalities, refer to the project licensing.
